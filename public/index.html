Build Instructions

1

Download Android Studio

Install Android Studio on your computer if you haven't already.

2

Download Project

Click the "Download Project" button above to get the ZIP file containing the complete Android project.

3

Open in Android Studio

Extract the ZIP file and open the project folder in Android Studio. Wait for Gradle sync to complete.

4

Generate Signed APK/Bundle

Go to Build > Generate Signed Bundle / APK. Create a new keystore or use existing one. Choose Android App Bundle for Google Play.

5

Upload to Google Play

Log in to Google Play Console, create a new app, and upload your signed AAB file.

Important: For Google Play, you must use an Android App Bundle (.aab) instead of APK. Keep your keystore file safe - you'll need the same keystore for all future updates!
